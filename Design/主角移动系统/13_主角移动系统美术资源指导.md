## 一、美术资源总览

### 1.1 资源需求清单

主角移动系统需要的美术资源主要包括以下类别：

| 类别 | 资源数量 | 优先级 | 说明 |
|------|---------|--------|------|
| 主角精灵图 | 约200帧 | 高 | 包含所有移动状态动画 |
| 武器精灵图 | 约100帧 | 高 | 不同武器的攻击动画 |
| 特效动画 | 约50个 | 中 | 闪避、格挡、攻击特效 |
| UI图标 | 约30个 | 中 | 耐力条、状态图标等 |
| 环境元素 | 若干 | 低 | 地面、阴影等支持元素 |

### 1.2 美术风格要求

根据项目美术风格指南（08_ArtStyleGuide），美术资源需满足：

- **分辨率**：主角32x32像素（单帧）
- **风格**：现代像素风，细节丰富
- **色调**：根据游戏整体调色板设计
- **帧率**：12 FPS（像素风格标准）

## 二、主角精灵图详细规格

### 2.1 主角基础规格

| 参数 | 数值 | 说明 |
|------|------|------|
| 画布尺寸 | 32x32像素 | 单帧画面尺寸 |
| 中心点 | (16, 16) | 角色中心位置 |
| 脚部位置 | y=28 | 地面接触点 |

## 七、推荐美术资源获取平台

### 7.1 免费资源平台

#### itch.io（推荐）

**特点**：
- 丰富的独立游戏资源
- 像素艺术资源量大
- 多为CC0或Creative Commons许可
- 可按风格筛选

**搜索关键词**：
- "pixel character"
- "2D character sprite"
- "pixel art animations"
- "adventure character"

**筛选建议**：
- 价格：Free
- 许可证：CC0 / Public Domain
- 风格：Pixel Art

#### OpenGameArt.org

**特点**：
- 完全免费的美术资源
- 社区活跃
- 资源类型全面
- CC0资源较多

**搜索关键词**：
- "character sprites"
- "2D animations"
- "pixel art"

#### Kenney.nl

**特点**：
- 高质量的免费资源
- 完全免费，可商用
- 风格统一
- 定期更新

**推荐资源包**：
- "Character Pack" 系列
- "Pixel RPG" 系列
- "Dungeon" 系列

#### Game Art 2D

**特点**：
- 专业的2D游戏美术
- 免费和付费资源都有
- 资源质量高
- 动画资源丰富

**搜索关键词**：
- "Free Character Sprite"
- "2D Animation Assets"

#### SpriteLib

**特点**：
- 开源精灵库
- 风格统一
- 免费使用
- 包含动画帧

### 7.2 付费资源平台（可选）

#### Itch.io（付费区）

- 高质量艺术家作品
- 支持创作者

#### Unity Asset Store

- 官方资源市场
- 与Unity集成度高
- 多种价格区间

#### GraphicRiver

- 专业设计师作品
- 单件购买
- 质量有保障

### 7.3 资源选择建议

#### 推荐获取方案

**方案A：完全免费方案**

1. **主角资源**：从itch.io下载免费像素角色包
2. **动画修改**：根据本文档规范调整动画帧数
3. **特效资源**：从OpenGameArt获取战斗特效
4. **UI资源**：使用Kenney.nl的UI资源包

**推荐资源包组合**：

| 资源类型 | 推荐来源 | 资源名称 | 预估帧数 |

## 八、资源制作规范

### 8.1 命名规范

#### 文件命名格式

```
[类型]_[角色]_[动作]_[方向]_[帧序号].[扩展名]

示例：
Player_Idle_Down_01.png
Player_Walk_Left_03.png
Player_Dodge_Roll_Up_05.png
```

#### 类型前缀

| 前缀 | 含义 |
|------|------|
| Player_ | 主角相关 |
| Weapon_ | 武器相关 |
| VFX_ | 视觉特效 |
| UI_ | 用户界面 |

### 8.2 导出规范

#### PNG导出设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 文件格式 | PNG-24 | 支持半透明 |
| 颜色模式 | RGB | 不需要Alpha通道时可只用RGB |
| 压缩 | 无压缩 | 保持像素清晰 |
| 尺寸 | 2的幂次 | 如32, 64, 128等 |

#### Sprite Sheet导出

如果使用精灵表（Sprite Sheet）：

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 布局 | 水平排列 | 便于Unity切割 |
| 间距 | 2像素 | 避免帧间干扰 |
| 边缘出血 | 1像素 | 防止渲染边缘锯齿 |

### 8.3 Unity导入设置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| Sprite Mode | Multiple | 多帧动画 |
| Pixels Per Unit | 32 | 与画布尺寸对应 |
| Filter Mode | Point (no filter) | 保持像素清晰 |
| Compression | None | 避免压缩失真 |
| Read/Write Enabled | 关闭 | 节省内存 |
| Generate Mip Maps | 关闭 | 像素游戏不需要 |

## 十一、常见问题

### Q1：找不到完全符合要求的资源怎么办？

**解决方案**：
1. 选择最接近的资源包
2. 后续可以使用Aseprite等工具进行修改
3. 先使用临时资源，后期替换为定制资源

### Q2：资源风格不一致怎么办？

**解决方案**：
1. 选择同一艺术家/工作室的资源包
2. 使用统一的滤镜处理
3. 后期统一调整配色

### Q3：动画帧数不够怎么办？

**解决方案**：
1. 使用插值技术增加中间帧
2. 在动画状态机中调整速度
3. 将长动画拆分使用

### Q4：资源文件过大怎么办？

**解决方案**：
1. 使用Sprite Atlas打包
2. 减少颜色数量（使用调色板）
3. 优化PNG压缩
