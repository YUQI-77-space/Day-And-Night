## 文档概述

本文档详细说明游戏中各类建筑的美术资源需求，包括外观设计、尺寸规格、结构细节等。建筑是小镇的重要组成部分，需要在像素风格中表现出末日后期的独特风格。

---

| 建筑类型 | 重要性 | 功能 | 美术复杂度 |

## 核心建筑详细设计

### 1. 市政厅（Town Hall）

#### 基础信息

| 项目 | 内容 |
|------|------|
| 建筑名称 | 市政厅 |
| 位置 | 中心广场北侧 |
| 功能 | 主线任务、镇长办公室 |
| 建筑风格 | 旧时代政府建筑 |
| 占地面积 | 3×2格（48×32像素） |

#### 外观设计

| 部位 | 设计说明 | 颜色参考 |
|------|----------|----------|
| 屋顶 | 红色瓦片屋顶，有修补痕迹 | #8B4513 / #A52A2A |
| 墙壁 | 白色灰泥墙，有污渍和裂缝 | #F5F5DC / #E8E8D0 |
| 门窗 | 木质门窗，油漆剥落 | #8B4513 / #654321 |
| 招牌 | "TOWN HALL"金属招牌 | #C0C0C0 / #1A1A1A |
| 台阶 | 石质台阶，磨损严重 | #808080 / #696969 |
| 装饰 | 旗帜（小镇旗帜）、旗杆 | #4169E1 / #C0C0C0 |

#### 设计特点

- **整体气质**：庄严但略显破旧，体现末日前的秩序感
- **特征**：屋顶有小镇旗帜飘扬，正门有镇长牌匾
- **细节**：墙壁上有弹孔或修补痕迹（表现动荡历史）
- **辨识度**：高，钟楼或高塔使其在远处可见

#### 结构分解

```
顶部结构：
- 屋脊：32×8像素
- 瓦片层：48×16像素
- 阁楼窗：16×16像素（可发光）

主体结构：
- 墙壁：48×32像素
- 窗户×2：16×16像素（可发光）
- 大门：16×24像素
- 招牌：32×8像素

基座结构：
- 台阶：48×8像素
- 基石：48×8像素
```

```

墙壁：32×24像素
  - 石砌纹理
  - 烟熏痕迹（上方）
  - 窗户（带铁栅栏）
  - 大门（宽大木门）

基座：32×8像素
  - 煤渣地面
  - 煤炭堆装饰
```

## 建筑状态变体

### 完好状态

- **特征**：建筑完整，功能正常
- **光照**：窗户透出温暖光线
- **细节**：干净整洁，无破损

### 受损状态

- **特征**：墙壁有裂缝、破损
- **光照**：部分窗户破碎或无光
- **细节**：修补痕迹明显

### 废弃状态

- **特征**：严重破损，杂草丛生
- **光照**：无灯光，一片漆黑
- **细节**：藤蔓覆盖，门窗破碎

## 文件命名规范

```
# 建筑外观
building_[建筑ID]_[状态].png
示例：
building_townhall_exterior.png
building_blacksmith_exterior.png
building_shop_exterior.png

# 建筑组件
building_wall_[类型]_[变体].png
building_roof_[类型]_[变体].png
building_door_[类型].png
building_window_[类型].png

# 建筑状态
building_[ID]_damaged.png   # 受损状态
building_[ID]_ruined.png    # 废弃状态
building_[ID]_night.png     # 夜晚状态

# 建筑装饰
building_sign_[类型].png     # 招牌
building_deco_[名称].png     # 装饰物

# 示例
building_townhall_damaged.png    # 受损的市政厅
building_roof_tile_red.png       # 红色瓦片屋顶
building_door_wood_ornate.png   # 华丽木门
```

### 设计原则

1. **辨识度高**：从远处能识别建筑功能
2. **风格统一**：所有建筑保持末日小镇风格
3. **细节丰富**：有生活痕迹和历史感
4. **模块化**：可灵活组合搭建不同建筑

## 注意事项

1. **模块化设计**：建筑组件要可自由组合
2. **一致性**：所有建筑保持风格统一
3. **可扩展性**：为未来建筑预留组件
4. **测试验证**：在Unity中实际测试搭建效果
5. **性能考虑**：避免过大的单张建筑图片
