## 文档概述

本文档详细说明游戏中武器的美术资源需求，包括近战武器、远程武器、法杖/魔法武器等各类武器的设计规范、动画需求和表现方式。武器是战斗系统的核心，需要在像素风格中表现出武器的力量感和独特性。

## 武器稀有度设计

### 普通（白）

- **特点**：基础外观，无特殊效果
- **示例**：生锈的铁剑、木弓
- **设计**：简洁实用，无装饰

### 优秀（绿）

- **特点**：有精良外观，有小幅加成
- **示例**：精钢剑、复合弓
- **设计**：有简单装饰线条

### 稀有（蓝）

- **特点**：优秀外观，有显著属性加成
- **示例**：精灵弓、炎魔杖
- **设计**：有宝石镶嵌

### 史诗（紫）

- **特点**：华丽外观，有强大特效
- **示例**：传说之剑、雷霆魔杖
- **设计**：复杂纹理+发光效果

### 传说（金）

- **特点**：终极外观，有炫目特效
- **示例**：传说之剑、传说之弓
- **设计**：全身发光+彩虹光效

## 文件命名规范

```
# 武器图标
weapon_[类型]_[名称].png
示例：
weapon_sword_rusty.png
weapon_bow_wood.png

# 武器精灵图（手持状态）
weapon_[类型]_[名称]_hand_[状态].png
示例：
weapon_sword_rusty_hand_idle.png
weapon_bow_wood_hand_fire.png

# 武器动画帧
weapon_[类型]_[名称]_anim_[帧数].png
示例：
weapon_sword_rusty_anim_01.png
weapon_bow_wood_anim_01.png

# 武器特效
vfx_weapon_[类型]_[特效名].png
示例：
vfx_weapon_sword_slash.png
vfx_weapon_fire_fireball.png

# 武器发光
glow_weapon_[名称].png
示例：
glow_weapon_legendary_sword.png

# 武器边框
border_weapon_[稀有度].png
示例：
border_weapon_rare.png
border_weapon_legendary.png
```

## 审核标准

### 设计标准

- [ ] 武器类型辨识度高
- [ ] 稀有度外观区分明显
- [ ] 攻击动画符合武器特性
- [ ] 特效与武器类型匹配

### 技术标准

- [ ] 尺寸符合规范（32×32）
- [ ] 方向处理正确
- [ ] 动画帧数达标
- [ ] 文件命名规范

### 性能标准

- [ ] 精灵图优化
- [ ] 特效粒子优化
- [ ] 内存占用合理

## 注意事项

1. **一致性**：所有武器保持风格统一
2. **可读性**：确保武器类型清楚可辨
3. **动画流畅**：攻击动画要有打击感
4. **特效适度**：不要影响游戏可玩性
5. **性能考虑**：特效粒子数量控制
