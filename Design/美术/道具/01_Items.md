## 文档概述

本文档详细说明游戏中道具的美术资源需求，包括道具图标、3D模型/精灵图、资源拾取物等。道具是玩家成长的重要组成部分，需要在有限的像素空间中清晰传达道具的功能和价值。

## 图标设计规范

### 基础规格

| 项目 | 规格 |
|------|------|
| 图标尺寸 | 16×16像素（图标格） / 32×32像素（放大显示） |
| Pixels Per Unit | 16 |
| 过滤模式 | Point（无模糊） |
| 透明背景 | 必须（PNG格式） |
| 边框 | 可选，1像素内缩 |

### 图标设计原则

1. **清晰可辨**：小尺寸下也能清楚识别
2. **色彩鲜明**：使用对比度高的颜色
3. **轮廓清晰**：边缘明确，无模糊像素
4. **风格统一**：所有图标保持像素风格一致
5. **功能暗示**：颜色和形状暗示道具类型

### 稀有度颜色标识

| 稀有度 | 颜色代码 | 边框颜色 | 用途 |
|--------|----------|----------|------|
| 普通 | 灰色 | 无/#808080 | 基础材料、普通消耗品 |
| 优秀 | 绿色 | #00FF00 | 精良装备、强力消耗品 |
| 稀有 | 蓝色 | #0088FF | 稀有装备、特殊道具 |
| 史诗 | 紫色 | #8800FF | 史诗装备、高级道具 |
| 传说 | 金色 | #FFD700 | 传说装备、终极道具 |

```
# 材料图标

# 货币图标
icon_currency_[类型].png

# 收藏品
icon_collectible_[名称].png

```

## 审核标准

### 基础标准

- [ ] 尺寸符合规范（16×16或32×32）
- [ ] 透明背景正确
- [ ] 像素边缘清晰
- [ ] 风格与游戏统一

### 设计标准

- [ ] 功能可识别
- [ ] 稀有度可区分
- [ ] 色彩搭配协调
- [ ] 细节适度

### 技术标准

- [ ] 文件命名规范
- [ ] 格式正确（PNG）

## 注意事项

1. **一致性**：所有道具图标风格统一
2. **可读性**：确保小尺寸下清晰可辨
3. **可扩展性**：预留命名空间和设计空间
4. **性能优化**：使用Sprite Atlas打包
5. **测试验证**：在游戏中实际测试显示效果
