> **文档版本：** v1.0

> **适用阶段：** Alpha阶段  
> **负责部门：** 美术部

## 文档概述

本文档详细说明游戏中消耗品的美术资源需求，包括生命恢复、状态恢复、临时强化、特殊消耗品等各类消耗品的设计规范和表现方式。消耗品是玩家生存的重要保障，需要在像素风格中清晰传达其功能和使用效果。

## 文件命名规范

```
# 消耗品图标
consumable_[类型]_[名称].png
示例：
consumable_potion_health_small.png
consumable_food_apple.png
consumable_bandage_basic.png
# 消耗品精灵图（世界掉落）
item_consumable_[名称].png
item_consumable_[名称]_anim.png
# 消耗品特效
vfx_consumable_[类型]_[名称].png
示例：
vfx_consumable_potion_health.png
vfx_consumable_buff_strength.png
# 消耗品边框
border_consumable_[稀有度].png
示例：
border_consumable_rare.png
border_consumable_epic.png
# 消耗品动画
anim_consumable_[名称]_[帧数].png
示例：
anim_consumable_health_potion_01.png
```

### 设计原则

1. **功能清晰**：一眼看出消耗品功能
2. **颜色编码**：不同类型用不同主色调
3. **品质区分**：稀有度有明确视觉差异
4. **动画反馈**：使用有明确的视觉反馈
5. **风格统一**：保持像素艺术风格一致

## 审核标准

### 设计标准

- [ ] 功能可识别
- [ ] 颜色搭配协调
- [ ] 稀有度区分明显
- [ ] 与游戏风格统一

### 技术标准

- [ ] 尺寸符合规范（16×16）
- [ ] 透明背景正确
- [ ] 动画帧数达标
- [ ] 文件命名规范

- [ ] 图标文件优化
- [ ] 特效粒子优化

## 注意事项

1. **一致性**：所有消耗品图标风格统一
2. **可读性**：小尺寸下也能识别功能
3. **区分度**：不同类型消耗品有明显区别
4. **反馈感**：使用时有明确的视觉反馈
5. **性能考虑**：特效粒子数量控制
