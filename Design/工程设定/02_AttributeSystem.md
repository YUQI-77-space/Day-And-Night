# 六维属性系统设计

## 系统概述

六维属性是本游戏的核心系统，深度影响玩法体验、剧情走向和战斗风格。

**六维属性：**
1. **智慧（Wisdom）**
2. **勇气（Courage）**
3. **力量（Strength）**
4. **技巧（Skill）**
5. **魅力（Charisma）**
6. **财富（Wealth）**

---

## 属性详细设计

### 1. 智慧（Wisdom）

**定义：** 知识储备、分析能力、策略思维

**主要提升方式：**
- 在图书馆阅读书籍
- 完成解谜类任务
- 研究敌人和物品
- 与学者型NPC交流

**影响范围：**

#### 白天影响
- 解锁特殊对话选项（识破谎言、提供建议）
- 某些任务需要智慧检定才能接取
- 制作高级物品时提高成功率
- 商店交易时识别物品真实价值

#### 夜晚战斗影响
- 显示敌人弱点信息
- 提高暴击率
- 解锁战术技能（如标记、分析）
- 陷阱伤害提升

#### 属性阈值效果

| 智慧值 | 解锁效果 |
|--------|----------|
| 10 | 基础敌人信息显示 |
| 20 | 解锁图书馆高级区域 |
| 30 | 识破NPC谎言，解锁侦探路线 |
| 40 | 制作科技类道具 |
| 50 | 预判敌人行动模式 |
| 60+ | 解锁真相路线关键线索 |

---

### 2. 勇气（Courage）

**定义：** 面对危险的决心、冒险精神、心理韧性

**主要提升方式：**
- 接受高风险任务
- 夜晚主动探索危险区域
- 击败强大敌人
- 在关键事件中做出勇敢选择

**影响范围：**

#### 白天影响
- 解锁高风险高回报任务
- 某些NPC只尊重勇敢的人
- 可以进入危险区域调查
- 影响某些剧情选择的可用性

#### 夜晚战斗影响
- 血量越低，伤害越高（狂战士效果）
- 提高格挡和反击成功率
- 减少受到的恐惧/眩晕效果时长
- 面对Boss时获得额外伤害加成

#### 属性阈值效果

| 勇气值 | 解锁效果 |
|--------|----------|
| 10 | 可以接受危险任务 |
| 20 | 夜晚可探索远郊区域 |
| 30 | 解锁狂战士技能树 |
| 40 | 单挑Boss时获得buff |
| 50 | 激励队友，提升团队战力 |
| 60+ | 解锁英雄路线 |

---

### 3. 力量（Strength）

**定义：** 身体力量、耐力、破坏力

**主要提升方式：**
- 在训练场锻炼
- 完成体力劳动（搬运、建造、采矿）
- 使用重型武器战斗
- 食用增强体质的食物

**影响范围：**

#### 白天影响
- 搬运更多物资
- 建造/修理速度提升
- 某些任务需要力量检定（破门、救人）
- 威慑某些NPC

#### 夜晚战斗影响
- 近战武器伤害提升
- 可以使用重型武器
- 击退/击倒敌人概率提升
- 破坏环境障碍物

#### 属性阈值效果

| 力量值 | 解锁效果 |
|--------|----------|
| 10 | 可使用中型武器 |
| 20 | 搬运量+50% |
| 30 | 可使用重型武器 |
| 40 | 解锁破坏者技能树 |
| 50 | 徒手也能造成可观伤害 |
| 60+ | 单次攻击可击退多个敌人 |

---

### 4. 技巧（Skill）

**定义：** 精准度、灵活性、技术熟练度

**主要提升方式：**
- 在工坊制作物品
- 练习射击/投掷
- 完成精细操作任务（修理、烹饪、开锁）
- 使用技巧型武器战斗

**影响范围：**

#### 白天影响
- 制作物品质量提升
- 修理成功率提高
- 开锁/解除陷阱
- 烹饪出更好的食物

#### 夜晚战斗影响
- 远程武器精准度和伤害
- 闪避率提升
- 连击数增加
- 暴击伤害倍率提升

#### 属性阈值效果

| 技巧值 | 解锁效果 |
|--------|----------|
| 10 | 基础制作配方 |
| 20 | 可以开锁 |
| 30 | 解锁刺客技能树 |
| 40 | 制作精良品质装备 |
| 50 | 完美闪避（闪避后短暂无敌） |
| 60+ | 制作传奇品质装备 |

---

### 5. 魅力（Charisma）

**定义：** 社交能力、说服力、领导力

**主要提升方式：**
- 与NPC互动交流
- 完成NPC委托任务
- 帮助小镇居民
- 参与社交活动

**影响范围：**

#### 白天影响
- NPC好感度提升速度加快
- 商店交易折扣
- 解锁特殊对话选项（说服、安慰、鼓励）
- 某些任务只有高魅力才能接取

#### 夜晚战斗影响
- 可以招募NPC同伴协助战斗
- 部分敌人可以被说服/混乱
- 同伴战斗力提升
- 获得支援技能

#### 属性阈值效果

| 魅力值 | 解锁效果 |
|--------|----------|
| 10 | 基础交易折扣 |
| 20 | 可招募1名同伴 |
| 30 | 解锁领袖路线 |
| 40 | 可招募2名同伴 |
| 50 | 说服敌人投降 |
| 60+ | 建立自己的势力 |

---

### 6. 财富（Wealth）

**定义：** 经济实力、资源积累、商业头脑

**主要提升方式：**
- 完成委托任务获得报酬

**影响范围：**

#### 白天影响

- 投资小镇建设，解锁新设施

- 贿赂某些NPC获取信息

#### 夜晚战斗影响
- 可以使用昂贵的消耗品
- 雇佣佣兵协助战斗
- 购买强力一次性武器
- 战斗前购买增益buff

| 财富值 | 解锁效果 |
|--------|----------|

| 50 | 购买稀有装备 |
| 60+ | 成为小镇首富，影响力巨大 |

## 属性成长机制

### 成长速度

- **初期（1-20天）：** 成长较快，每个属性容易提升到20-30
- **中期（21-50天）：** 成长放缓，需要专注特定属性
- **后期（51天+）：** 成长困难，需要完成特殊任务才能突破

### 成长上限

- 单个属性最高值：100
- 六维总和上限：300（一周目）
- 这意味着玩家必须做出选择，无法全属性满级

### 属性衰减机制

某些属性如果长期不使用会缓慢衰减：
- **力量：** 7天不训练/战斗，每天-1
- **技巧：** 7天不制作/练习，每天-1
- **其他属性：** 不会自然衰减

## 属性检定系统

### 检定触发时机

1. **对话选择时：** 某些对话选项需要属性达标
2. **任务接取时：** 高难度任务有属性门槛
3. **关键事件时：** 剧情分支点进行属性检定
4. **战斗中：** 某些技能需要属性支持

### 检定方式

**硬性检定：** 属性必须达到指定值
- 示例：智慧≥30才能接取侦探任务

**软性检定：** 属性越高成功率越高
- 示例：魅力影响说服成功率（魅力30=60%成功率，魅力50=90%成功率）

**组合检定：** 多个属性共同影响
- 示例：力量+技巧≥60才能使用某种特殊武器

## 技术实现要点

### 数据结构

```
PlayerAttributes {
  wisdom: int (0-100)
  courage: int (0-100)
  strength: int (0-100)
  skill: int (0-100)
  charisma: int (0-100)
  wealth: int (0-100)
  
  totalLimit: int (300 for first playthrough)
}
```

### 配置文件

所有属性相关的数值（阈值、成长速度、影响效果）都应该可配置，方便后期平衡性调整。

**示例配置：**
```json
{
  "attributes": {
    "wisdom": {
      "thresholds": [10, 20, 30, 40, 50, 60],
      "effects": {
        "10": "显示基础敌人信息",
        "20": "解锁图书馆高级区域",
        "30": "解锁侦探路线"
      },
      "growthRate": {
        "early": 1.0,
        "mid": 0.6,
        "late": 0.3
      }
    }
  }
}
```

### 事件系统集成

属性系统需要与事件系统深度集成：
- 事件触发时检查属性条件
- 事件结果影响属性成长
- 属性变化触发新事件

## 平衡性考虑

### 避免单一属性过强

- 每个属性都应该有独特价值
- 高难度内容需要多属性配合
- 某些Boss专门克制单一属性流派

### 鼓励多样化玩法

- 不同属性组合有不同的游戏体验
- 二周目可以尝试完全不同的build
- 隐藏内容需要特定属性组合才能触发

### 新手友好

- 初期不要求玩家立即做出选择
- 提供属性重置道具（稀有但可获得）
- 游戏内提供属性建议和指引

---

## 后续扩展

### 二周目+

- 提高总属性上限（300→400）
- 解锁新的阈值效果（70、80、90、100）
- 新的属性组合路线

### DLC可能性

- 新增第七维属性（如"信仰"）
- 属性觉醒系统（突破100上限）
- 属性转化机制（消耗一个属性强化另一个）
