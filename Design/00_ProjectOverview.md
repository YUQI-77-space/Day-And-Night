# 项目概览 - 末日小镇生存游戏

## 项目基本信息

**项目代号：** DayAndNight（暂定）  
**类型：** 2D像素风格 动作RPG + 肉鸽 + 养成  
**平台：** Windows / macOS  
**引擎：** Unity 2022 LTS  
**开发语言：** C#  
**目标受众：** 喜欢深度养成、多周目体验、末日题材的玩家

---

## 游戏核心概念（High Concept）

> 在末日废土的幸存者小镇中，玩家白天通过社交、任务和各种活动培养六维属性，夜晚则面对怪物的威胁进行动作战斗。玩家的属性成长将深刻影响剧情走向、任务类型和战斗风格，每个玩家都能体验到独特的游戏路线。

---

## 核心卖点（USP - Unique Selling Points）

1. **六维属性深度影响游戏体验**
   - 智慧、勇气、力量、技巧、魅力、财富
   - 不同属性组合解锁不同的任务路线和剧情分支
   - 属性直接影响战斗风格和可用策略

2. **昼夜双循环玩法**
   - 白天：自由探索、NPC互动、属性培养（类似《女神异闻录》）
   - 夜晚：动作战斗、肉鸽元素、资源掠夺

3. **末日废土世界观**
   - 压抑但不绝望的氛围
   - 丰富的NPC角色和小镇生态
   - 最终目标：拯救小镇，前往末日都市

4. **高重玩价值**
   - 多条主线路线
   - 不同属性搭配带来完全不同的游戏体验
   - 肉鸽元素保证每次战斗都有新鲜感

## 开发阶段规划

### Phase 0: 环境搭建（1-2周）
- Unity项目初始化
- Git版本控制
- 基础文档框架

### Phase 1: 核心原型（6-8周）
- **优先级最高：让角色动起来**
- 玩家移动和动画
- 基础战斗系统（攻击、闪避）
- 简单的敌人AI
- 对话系统原型
- 时间系统框架

### Phase 2: 系统扩展（8-12周）
- 六维属性系统
- NPC互动和好感度
- 任务系统
- 战斗系统深化
- 装备和道具系统

### Phase 3: 内容填充（持续迭代）
- 地图扩展（20+场景）
- NPC创建（50+角色）
- 主线剧情
- 支线任务
- 美术资源制作

### Phase 4: 优化与发布（4-6周）
- 性能优化
- 平衡性调整
- Bug修复
- 打包发布
